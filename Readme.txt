Razer Hydra DirectInput Wrapper

by Willi "Mr. Wonko" Schinmeyer

http://www.mrwonko.de

For Windows XP and up

== Usage ==

1. Download & Install PPJoy
   PPJoy Website: http://ppjoy.blogspot.com/ 
   Newest release at time of writing: http://ppjoy.blogspot.com/2010/03/new-test-release.html

2. Create Virtual Joysticks 1 - 4 in PPJoy

3. Start the Wrapper. Pray to get no errors.

3.1 Change the mappings in the Wrapper, if desired.

4. Have fun!
